## How CrUX field data fits in

Lab numbers above (PSI/Lighthouse) are from a single emulated run — they tell you how a page *can* perform on a clean device. GSC's CWV report uses **field** data from CrUX, which aggregates real Chrome user sessions over 28 days. Per-URL CrUX data only exists for pages with enough traffic; lower-traffic pages fall back to origin-level data (the "whole site" row at the top).
